List of Requirements for Test Distribution and Execution
--------------------------------------------------------

1.	It will be designed with BON
2.	It will make use of all available servers in parallel
3.	It will re-use Maven
4.	It will execute all existing test suites
5.	It will execute tests generated by external plug-ins such as RUT, KindOpts or JML
6.	It will maximise path coverage
7.  It will re-use Hudson
8. 	It will remember and re-use the tests from previous build for regression testing
9.  It will checkout the source code once for each local domain and share the local working copy
10.	It will authenticate will remote servers over an untrusted network e.g. using LDAP and SSH
11. If a required build tool or version is missing then copy it from another local server or install automatically
